# Weather API Project

A simple Python project to fetch and display real-time weather information for any city using the [**OpenWeatherMap API**](https://openweathermap.org).  


## Features

- Fetch current weather data for any city.
- Display temperature, humidity, weather condition, wind speed, sunrise, and sunset times.
- Includes emojis for different weather conditions.
- Supports **Metric, Imperial, and Standard** units.

## Installation

```bash
git clone https://github.com/enkoki/weather_api.git
cd weather_api
```

Optional: Create and Activate a virtual environment.

## Dependencies
requests
```bash
pip install requests
```
python-dotenv
```bash
pip install python-dotenv
```
